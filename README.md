## Installation

BXAutoRollLabel is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BXAutoRollLabel", :git => 'https://github.com/iException/BXAutoRollLabel.git'
```
